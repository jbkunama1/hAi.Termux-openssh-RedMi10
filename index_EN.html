📱 Termux OpenSSH Server on Android
(Tested on Redmi 10)

🚀 Complete Guide: Install Termux, OpenSSH & a script for easy start/stop/restart. SSH access to your Android phone from your local Wi‑Fi.

⭐ Visit GitHub Repo

🎯 Goal

  • ✅ Install Termux
  • ✅ Set up OpenSSH server
  • ✅ Script for start/stop/restart
  • ✅ Android settings (battery, network)

1️⃣ Install Termux

  1. Open F-Droid: f-droid.org (not Play Store!)
  2. Install Termux
  3. In Termux:
pkg update && pkg upgrade -y

2️⃣ Install OpenSSH server

pkg install openssh -y
passwd
whoami  # Remember username

Port: 8022 | Wake-lock: termux-wake-lock

3️⃣ Management Script (sshd-manage.sh)

Create the script:

curl -O https://raw.githubusercontent.com/jbkunama1/hAi.Termux-openssh-RedMi10/main/sshd-manage.sh
chmod +x sshd-manage.sh
mv sshd-manage.sh $PREFIX/bin/sshd-manage

Usage:

sshd-manage start
sshd-manage status
sshd-manage restart

4️⃣ Android Settings (Redmi 10)

SettingAction
Battery OptimizationTermux → No Restrictions
NetworkTermux → Allow Wi‑Fi/Mobile
AutostartSecurity App → Enable Termux

5️⃣ Find IP & SSH Connect

# IP in Termux
ifconfig wlan0 | grep inet

# From PC
ssh USER@192.168.1.45 -p 8022

🛠️ Troubleshooting

ErrorSolution
Connection timed outCheck IP/Wi‑Fi, Wake-lock
Permission deniedPassword/User/Port 8022
SSHD stopsDisable battery optimization

✅ Done! SSH access to your Android is ready 🚀
