📱 Termux OpenSSH Server auf Android
(getestet auf Redmi 10)

🚀 Vollständige Anleitung: Installiere Termux, OpenSSH & ein Script zum einfachen Starten/Stoppen/Restarten. SSH-Zugriff aus deinem WLAN auf dein Android-Smartphone.

⭐ GitHub Repo besuchen

🎯 Ziel

  • ✅ Termux installieren
  • ✅ OpenSSH-Server einrichten
  • ✅ Script für Start/Stop/Restart
  • ✅ Android-Einstellungen (Akku, Netzwerk)

1️⃣ Termux installieren

  1. Öffne F-Droid: f-droid.org (nicht Play Store!)
  2. Installiere Termux
  3. In Termux:
pkg update && pkg upgrade -y

2️⃣ OpenSSH-Server installieren

pkg install openssh -y
passwd
whoami  # Benutzername merken

Port: 8022 | Wake-Lock: termux-wake-lock

3️⃣ Verwaltungs-Script (sshd-manage.sh)

Erstelle das Script:

curl -O https://raw.githubusercontent.com/jbkunama1/hAi.Termux-openssh-RedMi10/main/sshd-manage.sh
chmod +x sshd-manage.sh
mv sshd-manage.sh $PREFIX/bin/sshd-manage

Nutzung:

sshd-manage start
sshd-manage status
sshd-manage restart

4️⃣ Android-Einstellungen (Redmi 10)

EinstellungAktion
Akku-OptimierungTermux → Keine Einschränkungen
NetzwerkTermux → WLAN/Mobilfunk erlauben
AutostartSecurity App → Termux aktivieren

5️⃣ IP finden & SSH verbinden

# IP in Termux
ifconfig wlan0 | grep inet

# Vom PC
ssh BENUTZER@192.168.1.45 -p 8022

🛠️ Troubleshooting

FehlerLösung
Connection timed outIP/WLAN prüfen, Wake-Lock
Permission deniedPasswort/User/Port 8022
SSHD stopptAkku-Optimierung deaktivieren

✅ Fertig! SSH-Zugriff auf dein Android ist bereit 🚀
